# **App Name**: SkinWise

## Core Features:

- Image Upload: Allow users to upload a photo of the affected skin area for analysis.
- Disease Classification: Use a pre-trained convolutional neural network (CNN) model to classify the uploaded image and detect potential skin diseases.
- Remedy Suggestion: Provide a list of possible remedies and treatments for the detected skin condition. A tool is used to determine whether a specific piece of information should be included in the model's generated response.
- Display Results: Display the classification results and suggested remedies in an easy to understand format.

## Style Guidelines:

- Primary color: Gentle sky blue (#87CEEB) for trustworthiness and calm.
- Background color: Soft off-white (#F8F8FF) to provide a clean, medical-like feel.
- Accent color: Muted green (#90EE90) to evoke health and healing; also for CTAs.
- Body and headline font: 'Inter' (sans-serif) for clear, modern readability.
- Use clear, easily recognizable icons to represent different diseases and remedies.
- Keep the layout clean and simple, focusing on ease of use and clear presentation of information.
- Subtle transitions and animations to improve user experience without being distracting.
